# Code of Conduct

## Our Standard

Clipfire is currently developed by a single founder with occasional
contributors. We expect everyone interacting with this codebase — whether by
reading it, opening an issue, submitting a pull request, or commenting on a
discussion — to behave professionally.

Specifically, we ask that you:

- Be respectful of other contributors and users.
- Assume good faith.
- Focus criticism on ideas, code, and design choices — not on people.
- Keep discussion on-topic for the issue or PR.

## Unacceptable Behavior

The following will result in your contribution being closed and, if
sustained, in your access to this repository being revoked:

- Personal attacks, harassment, or sustained disruption of discussion.
- Publication of others' private information without consent.
- Any form of discrimination on the basis of a protected characteristic.

## Reporting

If you observe behavior that violates this code, or if you believe you have
been the target of such behavior in connection with this project, please
email the maintainer privately:

[aconnor731@gmail.com](mailto:aconnor731@gmail.com)

Reports will be handled confidentially and as quickly as the maintainer's
schedule permits.

## Scope

This code of conduct applies in all project spaces — GitHub issues, pull
requests, code review comments, and any communication channels we publicly
link from this repository.
