# Reporting

If you have discovered a security vulnerability in Clipfire (formerly
Polemicyst), please email the address above with details. We commit to:

- Acknowledging your report within 72 hours.
- Investigating and confirming the issue.
- Coordinating disclosure with you before publishing a fix.

Please do not file public GitHub issues for vulnerabilities. Do not test
against production user data.
